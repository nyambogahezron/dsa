Welcome to your personal **DSA Mastery** repository! This project is a curated collection of core data structures and algorithms, implemented from scratch in TypeScript. It's designed to be your go-to reference for interview prep and deep dives into computer science fundamentals.

---

## Repository Structure

Each topic in this repository is self-contained with the following files:

- **`index.md`**: Theoretical notes, complexity analysis (Big O), and real-world use cases.
- **`index.ts`**: Clean, well-documented implementation of the data structure or algorithm.
- **`problems/`**: A collection of hand-picked interview questions to test your implementation.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed. We use `ts-node` to run TypeScript files directly.

### Running Implementation
To run or test a specific implementation, use:

```bash
# Example: Run the array implementation
npx ts-node data-structures/arrays/index.ts
```
