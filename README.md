# Data Structures and Algorithms in TypeScript

Welcome to your personal DSA learning repository! This project is structured to help you implement, understand, and master core data structures and algorithms using TypeScript.

## Getting Started

This repository is organized by data structure. Each topic has its own directory under `src/` with the following structure:

- **`index.md`**: Theoretical notes, time/space complexity analysis, use cases, and real-world applications.
- **`index.ts`**: The main implementation file where you write your code and test it with examples.
- **`problems/`**: A collection of at least 5 top interview problems with detailed descriptions and test cases to reinforce your understanding.

## How to Run

To run your implementation or test cases, you can use `ts-node`:

```bash
# Example: Run the array implementation
npx ts-node src/arrays/index.ts
```

## Practice Workflow

1. **Understand**: Read the `index.md` in a folder to learn about the data structure's theory and complexity.
2. **Implement**: Open `index.ts` and fill in the class methods.
3. **Practice**: Go to the `problems/` folder and solve the interview questions.
4. **Master**: Track your progress by updating the "Implementation Status" in each `index.md`.

Happy Coding! 💻
