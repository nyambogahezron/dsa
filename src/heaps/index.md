# Heaps

## Implementation Status
- [ ] Min-Heap/Max-Heap (Insert, Extract)

## Complexity Analysis
- **Insert**: O(log n)
- **Delete (extract min/max)**: O(log n)
- **Peek**: O(1)
- **Space**: O(n)

## Use Cases
- Priority queues.
- Heapsort algorithm.
- Finding the k-th smallest/largest element.

## Applications
- Operating system task scheduling.
- Graph algorithms (Dijkstra, Prim).

## Real World Usage
- Job scheduling in a background worker.
- Huffman coding for data compression.
