# Binary Indexed Trees (BIT)

## Implementation Status
- [ ] Update, Query (Prefix Sum)

## Complexity Analysis
- **Build**: O(n log n) or O(n)
- **Query (Prefix Sum)**: O(log n)
- **Update (Point)**: O(log n)
- **Space**: O(n)

## Use Cases
- Efficiently calculating prefix sums.
- Efficiently updating elements and querying prefix sums.

## Applications
- Frequency tracking.
- Inversion count in an array.

## Real World Usage
- Competitive programming.
- Spatial indexing.
